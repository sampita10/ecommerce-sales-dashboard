# 📊 Ecommerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project analyzes ecommerce sales data to understand customer behavior, sales trends, and profitability using Power BI.

---

## 🛠 Tools Used
- Power BI  
- DAX  
- Power Query (Data Cleaning)  
- Data Modeling  

---

## 📂 Dataset
The dataset contains information such as:
- Sales Amount  
- Quantity  
- Profit  
- State  
- Category  
- Payment Mode  

---

## 📊 Key Metrics
- Total Sales: 438K  
- Total Quantity: 5,615  
- Total Profit: 37K  
- Average Order Value: 121K  

---

## 🔍 Key Insights
- Clothing contributes the highest sales (63%)  
- Maharashtra leads in sales performance  
- COD is the most used payment method  
- Profit peaks in November  
- Printers and Bookcases generate highest profit  

---

## 📈 Dashboard Features
- Interactive slicers (State, Quarter)  
- KPI cards for quick insights  
- Sales trend analysis  
- Category and payment analysis
